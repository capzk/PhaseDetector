# PhaseDetector - 位面检测器

一个简单的魔兽世界插件，专门用于检测和通知位面(Phase)变化。

## 功能特性

- **自动位面检测**: 基于NPC GUID自动检测当前位面
- **智能通知**: 首次检测到位面时通知，位面变化时再次通知
- **全区域检测**: 不限制检测区域，所有地图都可检测（需要鼠标悬停或目标为NPC）
- **自动启动**: 插件加载后自动开始位面检测
- **多语言支持**: 支持简体中文、繁体中文、英文、俄文
- **通知方式**: 仅向自己发送系统消息（不发送聊天频道）
- **轻量级**: 无UI界面，纯命令行操作

## 安装方法

1. 将 `PhaseDetector` 文件夹复制到魔兽世界插件目录
2. 重启游戏或重新加载插件
3. 插件将自动开始检测位面

## 使用命令

- `/phd` 或 `/phasedetector` - 显示帮助信息
- `/phd on` - 启用插件
- `/phd off` - 禁用插件

## 工作原理

插件通过读取鼠标悬停或目标NPC的GUID来获取位面信息。位面ID由GUID的第3-4部分组成（分片ID-实例ID）。

## 技术说明

- **位面检测**: 使用 `UnitGUID()` API获取NPC的GUID
- **更新频率**: 鼠标指向NPC时即时检测位面变化
- **缓存机制**: 为每个地图缓存位面信息，避免重复通知
- **全区域支持**: 检测所有区域的位面变化，包括主城（需要鼠标悬停或目标为NPC）
## 注意事项

- 本插件不保存设置，重载或重登后会恢复默认配置。
